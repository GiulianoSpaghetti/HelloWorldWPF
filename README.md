# HelloWorldWPF
Un complesso Hello World Internazionalizzato in WPF

# Come funziona l'internazionalizzazione

L'internazionalizzazione non prende la lingua predefinita dell'interfaccia di sistema, perché al momento non c'è nessun modo sicuro di averla.
L'internazionalizzazione prende la prima lingua impostata nella lista delle lingue dell'utente e visualizza l'interfaccia in quella lingua.
Vi faccio vedere:
